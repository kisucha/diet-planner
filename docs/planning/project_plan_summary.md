# 프로젝트 계획 요약

이 문서는 자동 식단 생성 웹사이트 프로젝트의 전체 계획을 요약합니다.

## 1. 프로젝트 개요

사용자가 등록한 조리 가능한 음식을 기반으로 일일 식단을 자동으로 생성해주는 웹사이트를 개발합니다. 인터넷에서 조리 방법을 학습하고, 재료를 파악하며, 아침/점심/저녁/반찬/야식 등으로 구분하여 관리합니다.

## 2. 주요 기능

### 2.1 핵심 기능
- 레시피 등록 및 관리
- 식단 자동 생성
- 재료 비용 계산
- 달력 기반 시각화

### 2.2 부가 기능
- 인터넷 기반 레시피 학습
- 랜덤 식단 생성

## 3. 기술 스택

- **백엔드**: Node.js + Express.js
- **데이터베이스**: MariaDB
- **프론트엔드**: EJS 템플릿 엔진 + HTML/CSS/JavaScript
- **기타**: dotenv (환경변수 관리)

## 5. 개발 일정 (예상)

### 5.1 1단계: 기반 구조 구축 (3일)
- 데이터베이스 테이블 생성
- 기본 서버 설정
- 라우팅 및 컨트롤러 구현

### 5.2 2단계: 레시피 관리 기능 개발 (4일)
- 레시피 등록/조회/수정/삭제 기능
- 레시피 카테고리 분류
- API 엔드포인트 구현

### 5.3 3단계: 식단 생성 및 달력 기능 개발 (5일)
- 식단 생성 알고리즘 구현
- 달력 시각화
- 날짜별 식단 표시 기능

### 5.4 4단계: 재료 비용 계산 및 UI 개선 (3일)
- 재료 비용 계산 기능
- UI/UX 개선
- 반응형 디자인 적용

### 5.5 5단계: 테스트 및 최적화 (2일)
- 기능 테스트
- 성능 최적화
- 버그 수정

## 6. 품질 보증
